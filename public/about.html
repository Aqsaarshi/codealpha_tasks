🌟 What is MiniSocial?

MiniSocial is a lightweight social media platform created for learning and fun. Users can post short messages, upload images, like posts, and manage content.

👩‍💻 Developed By

This app was developed by Aqsa during her internship as a beginner-friendly project using HTML, CSS, JavaScript, and JSON Server.

💡 Features

  • Post messages with image support
  • Like, edit and delete posts
  • Search posts by name
  • Pagination with "Load More"
  • Responsive & modern design
